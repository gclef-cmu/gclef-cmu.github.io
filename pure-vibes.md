# Pure Vibes

**_Pure Vibes_ is a lightweight extension to [Pure Data](https://puredata.info) allowing for AI agent control**. It allows agents like Claude Desktop to help you create and understand patches. Try "Using Pure Vibes, create me an ambient sound synthesizer"!

## Get Started
- 📦 [**Install Pure Vibes**](https://github.com/gclef-cmu/pure-vibes#quick-start) by following the quick-start guide
- 💬 [**Leave feedback**](https://forms.gle/TRDoD46YURA8N41Z8) on our Google Form, or [open a GitHub issue](https://github.com/gclef-cmu/pure-vibes/issues)
- ⭐ [**GitHub repo**](https://github.com/gclef-cmu/pure-vibes): Pure Vibes is 100% open source!

> A mailing list is coming soon.
